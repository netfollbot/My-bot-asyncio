1. Введение
Данная Политика конфиденциальности описывает принципы работы приложения «Magist async» (далее — «Приложение»). Это Приложение является инструментом для строго личного использования, созданным его разработчиком для автоматизации задач в рамках собственной учетной записи Pinterest.

Основополагающий принцип: Приложение не собирает, не хранит, не обрабатывает и не передает каким-либо третьим сторонам данные пользователей.

2. Сбор и использование данных

· Отсутствие сбора данных: Приложение не имеет собственного сервера, базы данных или какого-либо механизма для сбора, хранения или передачи персональных данных пользователей.
· Доступ к Pinterest: Единственное взаимодействие с внешним сервисом осуществляется через официальный API Pinterest с использованием протокола OAuth 2.0. Это необходимо для безопасной аутентификации и выполнения операций (таких как закрепление, управление досками) от имени единственного и исключительно личного аккаунта разработчика.
· Данные учетной записи Pinterest: Токены доступа OAuth, необходимые для работы API, хранятся локально на устройстве, где запущено Приложение, и используются только для взаимодействия с API Pinterest. Эти данные не извлекаются, не копируются и не покидают среду выполнения Приложения.

3. Обмен данными с третьими сторонами
Приложение не передает данные каким-либо третьим сторонам. Оно взаимодействует исключительно с официальным API Pinterest для целей, описанных выше. Мы не продаем, не обмениваем и не передаем каким-либо иным способом какие-либо потенциальные данные.

4. Безопасность
Поскольку Приложение не собирает и не хранит пользовательские данные, вопросы централизованной защиты данных к нему не применимы. Безопасность данных учетной записи Pinterest обеспечивается стандартными мерами безопасности платформы Pinterest и безопасным хранением OAuth-токенов в локальной среде.

5. Использование исключительно в личных целях
Важно подчеркнуть, что данное Приложение не является публичным сервисом, коммерческим продуктом или SaaS-решением. Оно создано и используется одним человеком (разработчиком) для автоматизации рутинных задач в своем личном аккаунте Pinterest. Приложение не предназначено для использования другими лицами.
